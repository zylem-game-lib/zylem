# Zylem SpacetimeDB Server

This package is a placeholder for future SpacetimeDB multiplayer integration.

## Setup

To use SpacetimeDB with Zylem:

1. Install SpacetimeDB CLI: https://spacetimedb.com/install
2. Configure your server module in this directory
3. Update the scripts in `package.json` to use SpacetimeDB commands
